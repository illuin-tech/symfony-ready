Symfony-ready
=============

Docker images based on alpine 3 with php 7/8, node v12/14/16, composer 1/2 & following PHP exts:
- intl
- json
- mbstring
- pdo
- pdo_pgsql
- zip
- bcmath
- sockets
- ldap
- gd
