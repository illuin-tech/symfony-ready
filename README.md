Symfony-ready
=============

Docker images based on alpine 3.11 with php 7.3/7.4, node v12/14, composer & following PHP exts:
- intl
- json
- mbstring
- pdo
- pdo_pgsql
- zip
- bcmath
- sockets
- ldap
- gd


# Versions

| Tag                    | PHP | Composer | Node |
|------------------------|-----|----------|------|
| php7.3                 | 7.3 | 1        | 12   |
| php7.4                 | 7.4 | 1        | 12   |
| 7.4-composer-2-node-14 | 7.4 | 2        | 14   |
